# Talos node operations

Talos Linux, machine configs generated by **talhelper**. In `talos/`: `talconfig.yaml` (node defs), `talenv.yaml` (**source of truth** for `talosVersion`/`kubernetesVersion` ‚Äî docs are stale), `patches/`, `clusterconfig/` (generated + `talosconfig` for talosctl), `talsecret.sops.yaml` (never edit/decrypt).
